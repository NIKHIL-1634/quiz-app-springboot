# Quiz App - Spring Boot

## Tech Stack
- Spring Boot
- Spring Data JPA
- PostgreSQL
- REST APIs
- Simple Frontend (HTML + JS)

## Features
- Add Questions
- Create Quiz by Category
- Random Question Selection
- Submit Quiz
- Auto Score Calculation
- Show Correct Answers

## How to Run
1. Configure PostgreSQL in application.properties
2. Run QuizAppApplication
3. Open http://localhost:8080
